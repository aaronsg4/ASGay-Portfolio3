This is the fizzBuzz exercise

Enter two values between 1 and 100 below to start the fizzBuzz exercise

When you hit submit, the computer will find each number between 1-100 that is divisible by your first number and replace it with fizz. Then it will find each number between 1-100 that is divisible by your second number and replace it with buzz. If a number is divisible by BOTH of your numbers, the computer will write fizzBuzz!
